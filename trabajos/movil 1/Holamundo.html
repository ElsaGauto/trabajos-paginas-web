MDN Web Docs es una plataforma de aprendizaje para tecnologías web y el software que impulsa la web.

El javascript es el lenguaje de programación que se utiliza principalmente para crear páginas web dinámicas. Es una página web dinámica es aquella que incorpora los efectos como texto que aparece y desaparece, animaciones, acciones que se activan al pulsar unos botones y ventajas con mensajes de aviso del usuarios.
